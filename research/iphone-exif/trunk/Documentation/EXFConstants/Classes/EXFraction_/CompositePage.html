EXFraction

A simple fraction class used to store all rational data types

Superclass: NSObject
Declared In: EXFConstants.h

Discussion

The fraction class is used to avoid precision issues when converting from the fraction format stored in the JPEG image. The image data is stored as two longs (numerator and denominator)
